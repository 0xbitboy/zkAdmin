# zkAdmin(开发中.......)
一个zookeeper管理工具，主要支持节点的增删查改等功能。

- zkAdmin-api是后端的项目,用的是springboot+shiro的架构，zookeeper客户端用的是官方的客户端。
- zkAdmin-UI 是前端的项目，打算用Vue-element来做（正在学习中）
